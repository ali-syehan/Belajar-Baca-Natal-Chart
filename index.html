Belajar Membaca Natal Chart
Klasik + Modern · eBook · Guiding · Input manual/semi-offline

Belajar membaca natal chart dari laporan chart yang sudah Anda punya.

Aplikasi ini dibuat untuk pembelajar yang sulit menghafal. Hafalan dibangun lewat praktik berulang: melihat simbol, klik istilah, mengikuti formula, lalu menyusun interpretasi.

Klasik / TradisionalModernGuiding aktifSimbol visualOffline HTML

Alur aplikasi

1. Masukkan data chart satu kali di halaman Input Data Chart.

2. Data itu mengisi dua halaman: Natal Chart Klasik dan Natal Chart Modern.

3. Jika data modern belum lengkap, halaman Modern memberi tahu bagian yang perlu diisi.

4. Guiding memberi penjelasan cepat, lalu link ke materi lengkap dengan tombol BACK untuk kembali.

Input Data Chart

Masukkan data satu kali. Setelah diproses, data mengisi halaman Klasik dan Modern.

Contoh Chart Latihan

Pilih contoh yang ingin dibaca. Nama asli disembunyikan menjadi Person A, Person B, dan seterusnya.
Catatan: contoh ini untuk latihan membaca. Data yang tampil di aplikasi hanya kode Person. Untuk Person A, C, D, dan K, data Chiron belum dihitung di contoh bawaan; halaman Modern akan memberi input tambahan jika Anda ingin melengkapinya.

Cari Data Natal Chart / Semi-Offline

Buat prompt, cari data di ChatGPT lain, lalu paste JSON.

Paste JSON Natal Chart

JSON dari prompt semi-offline akan diproses menjadi data aplikasi.

Input Manual dari Laporan Natal Chart

Gunakan jika Anda sudah punya data house, planet, dan aspek.
Isi 12 house dari laporan chart. Detik tidak wajib. Simbol zodiak muncul agar mudah hafal.
Untuk pembacaan Klasik, 7 planet klasik adalah data utama.
Bagian Modern. Jika Anda hanya membaca Klasik, bagian ini boleh dikosongkan. Jika membuka Natal Chart Modern, data yang kurang akan diberi tahu.
Aspek opsional. Jika belum tahu applying/separating, pilih “Tidak tahu”.
