# Belajar Membaca Natal Chart — Klasik & Modern

Ini adalah paket siap upload ke GitHub Pages.

## Isi paket

- `index.html` — file HTML tunggal aplikasi.
- `.nojekyll` — agar GitHub Pages tidak memproses file sebagai Jekyll site.
- `README.md` — keterangan repository.
- `PANDUAN_GITHUB_PAGES.md` — panduan upload dan aktivasi GitHub Pages.

## Cara cepat

1. Buat repository baru di GitHub.
2. Upload semua file dari ZIP ini ke root repository.
3. Buka **Settings → Pages**.
4. Pada **Build and deployment**, pilih **Deploy from a branch**.
5. Pilih branch **main** dan folder **/root**.
6. Klik **Save**.
7. Tunggu sampai muncul link GitHub Pages.

Format link biasanya:

```text
https://USERNAME.github.io/NAMA-REPOSITORY/
```
